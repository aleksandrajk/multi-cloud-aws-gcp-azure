# **Multi-Cloud Deployment with Terraform**  
Deploys a **scalable web app** on **AWS, GCP, and Azure** using Terraform.  

## **🚀 Features**  
✅ **AWS**: EC2 + S3  
✅ **GCP**: Compute Engine + Cloud Storage  
✅ **Azure**: VM + Blob Storage  

## **⚙️ Prerequisites**  
- Terraform (`v1.0+`)  
- AWS/GCP/Azure accounts  
- SSH key pairs
